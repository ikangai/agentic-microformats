# Contributing to Agentic Microformats

Thank you for your interest in contributing to the Agentic Microformats specification!

## Ways to Contribute

### Report Issues

Found a problem or inconsistency in the specification? Please open an issue with:

- A clear description of the problem
- The section of the spec affected
- A suggested fix (if you have one)

### Share Implementation Experience

If you've implemented Agentic Microformats in a site or agent, we'd love to hear about it:

- What worked well?
- What was confusing or difficult?
- What's missing from the spec?

Open a Discussion to share your experience.

### Propose Changes

For changes to the specification:

1. Open an issue first to discuss the change
2. Fork the repository
3. Make your changes to `SPECIFICATION.md`
4. Submit a pull request with a clear description

### Add Examples

Good examples help people understand the spec. To add an example:

1. Create an HTML file in the `examples/` directory
2. Include comments explaining the annotations
3. Follow the style of existing examples
4. Submit a pull request

## Style Guidelines

### Specification

- Use RFC 2119 keywords (MUST, SHOULD, MAY) consistently
- Include concrete examples for new concepts
- Keep backward compatibility in mind
- Document breaking changes clearly

### Examples

- Use semantic, readable HTML
- Include CSS for visual clarity
- Add comments explaining agent-relevant features
- Test that the HTML is valid

### Commit Messages

- Use present tense ("Add feature" not "Added feature")
- Reference issues when relevant ("Fix #12: clarify trust regions")
- Keep the first line under 72 characters

## Versioning

The specification follows semantic versioning:

- **Patch** (0.1.x): Clarifications, typo fixes, non-breaking additions
- **Minor** (0.x.0): New features, backward-compatible changes
- **Major** (x.0.0): Breaking changes

## Code of Conduct

Be respectful and constructive. This is a collaborative project aimed at improving human-AI interaction on the web.

## Questions?

Open a Discussion if you're unsure about anything. We're happy to help!
